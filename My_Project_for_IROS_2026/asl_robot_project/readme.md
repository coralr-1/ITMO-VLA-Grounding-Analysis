# 📘 Shadow Hand AI 语音控制系统 - 用户使用手册

## 1. 系统简介

本项目是一个基于人工智能的机械手控制系统。它能“听懂”你说的话，智能纠正口误，翻译成英文，并控制仿真机械手（Shadow Hand）打出对应的 ASL 手语动作。

**核心功能：**

* 🎤 **语音识别**：支持中文语音输入。
* 🧠 **智能纠错**：利用大模型自动修复同音错别字（如把“怎么周”修复为“怎么走”）。
* 🤖 **闭环控制**：动作平滑，自动检测关节是否到位，杜绝鬼畜。

---

## 2. 环境准备 (Prerequisites)

在运行之前，请确保电脑满足以下条件：

* **操作系统**：Windows 10/11 或 Linux (Ubuntu)。
* **Python 版本**：Python 3.10 或更新版本。
* **必要软件**：[LM Studio](https://lmstudio.ai/) (用于运行本地大模型)。
* **硬件**：麦克风（必须）、互联网连接（用于 Google 语音识别 API）。

---

## 3. 安装步骤 (Installation)

### 第一步：准备文件

确保以下 3 个文件在**同一个文件夹**内：

1. `asl_main.py` (主程序)
2. `my_shadow_hand.urdf` (机械手模型)
3. `requirements.txt` (依赖列表)

### 第二步：安装 Python 依赖

在文件夹空白处 **右键 -> 在终端中打开 (Open in Terminal)**，输入以下命令并回车：

```bash
pip install -r requirements.txt

```

> **⚠️ Windows 用户特别注意：**
> 如果 `pyaudio` 安装失败（报错红色文字），请执行以下两行补救命令：
> 1. `pip install pipwin`
> 2. `pipwin install pyaudio`
> 
> 

### 第三步：配置“大脑” (LM Studio)

1. 打开 **LM Studio** 软件。
2. 在搜索栏搜索并下载模型：推荐 **Qwen2.5-1.5B-Instruct** (速度快) 或 **Qwen2.5-7B-Instruct** (更聪明)。
3. 点击左侧的 **Local Server** (双向箭头图标)。
4. 点击顶部的绿色按钮 **Start Server**。
5. 确保日志显示 `Server running at http://localhost:1234`。

## 4. 运行与操作 (How to Run)

### 启动程序

在终端中输入：

```bash
python asl_main.py

```

当看到 `=== Shadow Hand 智能控制系统 ===` 和 `操作指南` 时，说明启动成功。

### 操作模式

#### 模式 A：语音控制 (推荐 ✨)

1. 在终端输入字母 **`v`** 并按回车。
2. 屏幕提示 `🎤 正在调整环境噪音...` 时，**请保持安静 1 秒钟**。
3. 屏幕出现 `🔴 请说话! (Listening...)` 时，对着麦克风说出中文句子。
* *示例：“你好我想去学校”*


4. 说完后稍等片刻，系统会自动识别、纠错并开始动作。

#### 模式 B：文字调试

1. 直接在终端输入中文或英文句子，按回车。
* *示例：“Hello World” 或 “你好”*


2. 机械手将直接执行。

#### 退出系统

* 输入 **`q`** 并按回车即可关闭。

## 6. 常见问题 (FAQ)

**Q1: 报错 `ModuleNotFoundError: No module named 'speech_recognition'**`

* **A:** 你忘了安装依赖库。请运行 `pip install SpeechRecognition`。

**Q2: 语音识别一直显示“超时”或“无法识别”？**

* **A:**
1. 确保你的电脑连上了互联网（需要连接 Google API）。
2. 检查麦克风是否静音。
3. 说话声音大一点，稍微慢一点。



**Q3: 机械手不动，或者报错 `Connection Error`？**

* **A:** 检查 LM Studio 是否开启了 **Start Server**，且端口号必须是 `1234`。

**Q4: 机械手动作太慢？**

* **A:** 这是正常的。为了防止机械手“鬼畜”或穿模，我们开启了**闭环反馈控制**，确保每个手指都弯曲到位才会做下一个动作。这比单纯的“快”看起来更真实、更优雅。
